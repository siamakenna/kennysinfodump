# Makenna Rodriguez e-Portfolio

Static portfolio site built for LinkedIn, GitHub, and GitHub Pages.

## Files

- `index.html` - Portfolio content and page structure.
- `styles.css` - Responsive visual design.
- `script.js` - Mobile menu, sticky header state, routed sections, hidden gems, and footer year.
- `assets/` - Downloaded public imagery used by the site.

## Publish on GitHub Pages

1. Push this folder to a GitHub repository.
2. In GitHub, open the repository settings.
3. Go to **Pages**.
4. Set the source to the `main` branch and root folder.
5. Save and use the generated GitHub Pages URL on LinkedIn.

The site is intentionally build-free, so GitHub Pages can serve it directly.
